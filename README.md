## 1. 개 요
**작성방향 : 왜 이문제를 해결하고자 하는가? 이 문제를 해결하는 것이 어떠한 의미가 있는가? 무엇을 확인해보고 싶은가?**

영화는 가장 전형적이고 보편적인 문화활동 중 하나로, 대중에게 대단히 큰 전파력을 가지는 핵심 콘텐츠입니다. 디지털 시대에 접어들어서는 다양한 온라인 영화 평론 및 리뷰 플랫폼이 구축되었으며, 실시간으로 축적되는 관객들의 평가는 영화의 장기 흥행과 직결되는 핵심 척도로 활용되고 있습니다.

본 프로젝트에서는 국내외 과학계와 대중의 엄청난 관심을 받으며 국내 천만 관객 돌파 및 글로벌 7억 달러 흥행을 달성한 크리스토퍼 놀란 감독의 SF 대작 영화 **'인터스텔라(Interstellar)'**에 대한 한국어 리뷰 감성을 분석하고자 합니다. 인터스텔라는 경이로운 비주얼과 압도적인 흥행을 기록한 동시에, 작품이 전개하는 상대성 이론, 블랙홀, 웜홀 등 난해한 물리 법칙 지식과 3시간에 육박하는 러닝타임으로 인해 관객들 사이에서 매우 뚜렷한 호불호 흐름을 형성했습니다. 

따라서 본 프로젝트에서는 오픈소스 플랫폼인 GitHub 오픈 저장소로부터 확보한 한국어 영화 평론 데이터셋(`ratings.txt`)을 기반으로 미세조정(Fine-tuning)된 다국어 BERT 모델을 구축하여 관객의 긍정·부정 반응을 정밀하게 분류 및 예측합니다. 이를 통해 최종적으로 불호(부정적 리뷰) 반응을 보인 관객층의 핵심 불만 이슈를 명확히 정량화하여 분석하는 것을 목표로 합니다.

## 2. 데이터 (Dataset)
### 2.1 데이터 수집 및 명세
- **데이터 출처**: 오픈소스 개발 플랫폼 GitHub의 오픈 데이터셋 저장소([e9t/nsmc](https://github.com/e9t/nsmc))에 공개 공유되어 있는 공신력 있는 대규모 한국어 영화 평론 텍스트 코퍼스 파일인 **`ratings.txt`**를 파이프라인의 원천 소스로 활용하여 수집하였습니다.
- **데이터 구성 항목**: 데이터는 영화 리뷰마다 고유하게 부여된 고유 식별자(`id`), 사용자가 입력한 한국어 리뷰 문장 원문(`document`), 그리고 데이터 수집 시점에 사전 할당된 감성 정답 가이드 라벨(`label`, 0: 부정 / 1: 긍정)로 명확히 구성되어 있습니다.
- **품질 검증 및 안전 장치**: 텍스트 데이터의 파일 전처리 과정 중 빈 값(`NaN`)이 포함되어 있으면 토큰화(Tokenization) 연산 시 치명적인 구문 파싱 예외가 발생합니다. 파이프라인의 안전성을 최고조로 확보하기 위해 로드 직후 `df.dropna(subset=["document", "label"])` 공정을 강제 적용하여 결측치를 원천 배제합니다.

### 2.2 탐색적 데이터 분석 (EDA)
GitHub로부터 수집된 데이터셋은 아래 표 구조와 같이 탭(`\t`) 기호를 구분자로 삼아 체계적으로 구조화되어 있습니다.

|구분|피처명 (Column)|실제 데이터 구조 및 예시|
|---|---|---|
|고유 식별자|id|8112052|
|입력 데이터|document|"어릴때보고 지금다시봐도 재밌어요ㅋㅋ" / "지루하고 난해해서 중간에 졸았습니다..."|
|정답 라벨|label|1 (긍정적 반응) 또는 0 (부정적 반응)|

## 3. 학습 데이터 및 파이프라인 구축

### 3.1 학습 데이터 구축 전략
- **추출 및 구축 전략**: 원본 데이터셋이 보존하고 있는 현실 세계의 가중치 분포와 호불호 비율을 인위적 왜곡 없이 완벽하게 투영하기 위해 인위적인 조정 요소를 완전 배제하고 무작위 샘플링(Random Sampling)을 적용하여 학습 소스를 정제합니다.
- **실험 데이터 스케일**: 최적의 파인튜닝 수렴 연산 효율과 연산 자원을 감안하여 최대 학습 샘플 규모를 **`3,000건(MAX_SAMPLES = 3000)`**으로 규정하여 최종 실험 서브 데이터셋을 가공합니다.
- **실험 재현성**: 무작위 셔플링의 시드 난수 상태를 상시 통제하기 위하여 `RANDOM_STATE = 2026`을 고정 적용합니다.

## 4. BERT 모델 학습 및 미세조정 (Fine-tuning)

- **데이터셋 격리 분할**: 텍스트 인코딩 마감 이후 전체 데이터 구조를 **학습 데이터 80%(2,400건)**와 **검증 데이터 20%(600건)**의 스케일로 철저히 격리 분할하여 모델의 만성적인 취약점인 과적합(Overfitting) 현상을 예방합니다.
- **한국어 전용 인코딩**: 다국어 말뭉치 및 국문 텍스트 내 자소 임베딩 특성을 치밀하게 수용하는 사전 학습 인공지능 모델인 **`bert-base-multilingual-cased`** 및 전용 토크나이저를 채택합니다. 평론 특성상 짧은 단평 중심의 발화 구조가 지배적이므로 메모리 오버헤드를 막고 추론 속도를 높이기 위해 문장 제한 최대 가이드 라인을 **`MAX_LENGTH = 128`**로 최적화 지정합니다.
- **배치 다이렉션**: 연산 기기의 VRAM 버퍼 터짐을 막기 위해 `TensorDataset`과 `DataLoader` 인프라를 바인딩하여 미니배치 규격을 **`BATCH_SIZE = 8`** 스케일로 모델에 쪼개어 피딩합니다. 학습 세션에는 무작위 패턴의 `RandomSampler`를, 검증 세션에는 순차적 연산의 `SequentialSampler`를 주입합니다.
- **가중치 제어 최적화**: 손실 가중치 오차 보정을 위해 가중치 감쇠 알고리즘이 포함된 `AdamW(lr=2e-5, weight_decay=0.01)` 옵티마이저를 셋업합니다. 아울러 초기 10% 학습 지점까지의 단계적 예열로 가중치 폭발을 제어하는 선형 웜업 스케줄러(`get_linear_schedule_with_warmup`) 및 임계 그래디언트 클리핑(`max_norm=1.0`)을 적용합니다.

### 4.1 [시각화 2] 에포크별 모델 성능 실시간 학습 곡선 지표 (Training Metrics)
총 4 에포크(EPOCH) 동안 훈련 모델이 최적점에 도달할 수 있도록 미세조정 루프를 전개하며, 매 에포크마다 훈련 손실값(Loss), 학습 데이터 정확도(Training Accuracy), 검증 데이터 정확도(Validation Accuracy)를 엄격히 로깅합니다. 

루프 종료 시 코드에 연동된 `plot_training_results` 엔진을 통해 에포크별 손실 감쇠 곡선과 학습 vs 검증 정확도의 추세를 한눈에 평가할 수 있는 통합 지표 선 그래프 그래프를 **`plots/02_training_metrics.png`** 경로에 자동으로 생성 및 저장합니다.

## 5. 문제제기에 대한 결과 및 불호 원인 분석 연계
- **감성 매트릭스 도출**: 완전하게 최적 파인튜닝이 종결된 다국어 BERT 시퀀스 분류기를 활용하여 인터스텔라 도메인에 대한 전반적인 관객 리뷰 텍스트 데이터의 긍/부정 분포 스코어를 예측 집계합니다.
- **텍스트 마이닝 확장 체계**: 본 분류 자동화 모델이 명확하게 불호 반응인 부정 라벨 (`label: 0`)로 확정 분류해 낸 문장들만을 시스템적으로 정밀 필터링하여 추후 설계될 **토픽 모델링(Topic Modeling: LDA 또는 BERTopic)** 분석 시스템에 전달합니다. 이를 통해 관객들이 작품에 대해 불만을 제기한 구체적 이슈 키워드(예: 난해한 물리 법칙, 긴 러닝타임, 초반 전개의 지루함 등)의 통계적 결합 가중치를 도출하고 정량 분석 지표를 마감합니다.

## 6. 느낀점 및 개선방향
- **경량 파인튜닝의 유효성 체감**: GitHub에서 얻어온 한국어 영화 리뷰 중 3,000건의 샘플과 4 에포크 학습 가동만 진행했음에도 트랜스포머 아키텍처 특유의 문맥 맥락 간 파악 성능 덕분에 검증 셋에서 대단히 고무적이고 높은 수준의 수렴 정확도를 조기에 포착할 수 있었습니다.
- **향후 개선 과제**:
  1. 다국어 기반의 BERT 모델(`bert-base-multilingual-cased`)도 우수한 안정성을 입증했으나, 국내 영화 커뮤니티 평론 텍스트 도메인에 고유하게 관측되는 한국어 구어체, 이모티콘, 독창적인 신조어 구조의 뉘앙스를 한 치의 오차 없이 판별하기 위해 향후 **KcBERT** 또는 **KoELECTRA** 같은 한국어 특화 타깃 언어 모델로 아키텍처를 업그레이드할 계획입니다.
  2. 긍부정 중심의 단순한 이진 분류(0과 1) 영역을 넘어, 영화 관객들이 텍스트 내에 직접 내포한 감정의 파동을 경이로움, 감동, 지루함, 실망 등 다중 클래스 분류(Multi-class Classification) 구조로 정밀화하여 호불호의 근본적 원인을 보다 다차원적으로 파헤치는 고도화 과제를 수행할 예정입니다.
